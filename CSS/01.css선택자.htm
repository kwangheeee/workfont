css 선택자

선택자란 특정 html 요소를 선택할 때 사용하는 기능으로
요소를 선택하여 원하는 스타일과 기능을 적용할 수 있다

모든 선택자

html 문서 안의 모든 태그를 선택할 떄 전체 선택자를 사용한다 
        *{
            스타일 속성:값;
            스타일 속성:값;
            ...
        }

태그 선택자(태그명)

태그명을 이용해 요소를 선택 
        태그명{
            스타일 속성:값;
            스타일 속성:값;
            ...
        }

아이디 선택자(#아이디)

요소의 id속성을 이용하여 요소 선택
#아이디 속성값 
        #아이디명{
            스타일 속성:값;
            스타일 속성:값;
            ...
        }

클래스 선택자(.클래스명)

html 문서 내에 여러 개의 태그를 동일한 클래스명으로 지정하여 선택할 때 사용 
        .클래스명{
            스타일 속성:값;
            스타일 속성:값;
            ...
        }
